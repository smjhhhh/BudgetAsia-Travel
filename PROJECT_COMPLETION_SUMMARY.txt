================================================================================
                         DESIGN SYSTEM
================================================================================

RESPONSIVE BREAKPOINTS:
- Mobile (default): < 640px (smartphones)
- Tablet (md:): ≥ 768px (iPad, tablets)
- Desktop (lg:): ≥ 1024px (monitors)

COLOR SCHEME:
City-Specific Colors:
├── Bangkok: Amber (#FBBF24) for warm, vibrant aesthetic
├── Tokyo: Pink (#EC4899) for modern, trendy feel
├── Seoul: Blue (#2563EB) for cool, contemporary vibe
└── Hong Kong: Purple (#A855F7) for cinematic, sophisticated look

Global Colors:
├── Primary CTA: Emerald-700 (#047857) - "Book Now" buttons
├── Text: Zinc-900 & Zinc-700 for readability
├── Accents: White/Gray for contrast
├── Backgrounds: White/90 opacity for depth

TYPOGRAPHY:
- Headings: Font Light (weight: 300) for modern elegance
- Body Text: Font Light/Regular for readability
- Tracking: Tight tracking on headings for sophistication

EFFECTS & ANIMATIONS:
- Hover Scales: 1.05x on images (500ms transition)
- Color Transitions: 200ms smooth color changes
- Opacity Effects: bg-white/90 for layered depth
- Backdrop Blur: Navigation sticky bar (blur-md)
- Animations: Spinning record player, bouncing scroll indicator
- Shadows: Subtle shadow-sm for depth

SPACING SYSTEM:
- Container: max-width-6xl with px-8 padding
- Sections: py-24 for consistent vertical spacing
- Cards: gap-8 (attractions), gap-6 (food)
- Typography: mb-4, mb-6 for heading spacing
