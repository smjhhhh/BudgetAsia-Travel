## Project Overview

BudgetAsia Travel is a comprehensive travel planning platform designed to help users discover and book affordable travel experiences across major Asian destinations including Bangkok, Tokyo, and Seoul.

## Tech Stack

- **HTML5:** Semantic markup for accessibility and SEO
- **TailwindCSS:** Utility-first CSS framework (CDN)
- **JavaScript:** Vanilla ES6+ for interactive features
- **Git:** Version control with feature-branch workflow

## Site Structure

```
/
├── index.html          # Homepage with destination highlights
├── bangkok.html        # Bangkok destination details
├── tokyo.html          # Tokyo destination details
├── seoul.html          # Seoul destination details
├── packages.html       # Travel package listings
├── booking.html        # Interactive booking form
└── reviews.html        # Customer testimonials
```

## Features

- **Responsive Design:** Mobile-first approach using TailwindCSS
- **Dynamic Booking System:** Client-side form validation and DOM manipulation
- **Semantic HTML:** Accessible and SEO-optimized structure
- **Component-Based Architecture:** Reusable navigation and footer components

## Development Workflow

This project follows a **feature-branch workflow**:

1. Create feature branch from `main`
2. Implement feature with atomic commits
3. Create Pull Request with detailed description
4. Merge after review

### Commit Message Convention

- `feat:` New feature implementation
- `style:` CSS/design changes
- `fix:` Bug fixes
- `docs:` Documentation updates
- `refactor:` Code restructuring

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/smjhhhh/BudgetAsia-Travel.git
   ```

2. Open `index.html` in your browser (no build process required)

3. For development:
   ```bash
   # Create a feature branch
   git checkout -b feature/your-feature-name

   # Make changes and commit
   git add .
   git commit -m "feat: your feature description"

   # Push to remote
   git push origin feature/your-feature-name
   ```
